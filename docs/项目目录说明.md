各目录职责：
- `backend/src/config`: 环境变量、数据库配置、后端运行参数。
- `backend/src/routes`: 路由定义，只负责 URL 到 controller 的映射。
- `backend/src/controllers`: 接收请求、调用 service、返回响应。
- `backend/src/services`: 业务编排层。`maintenanceService.js` 负责业务规则，`maintenanceChainService.js` 负责链上调用。
- `backend/src/models`: 数据访问层，直接操作 MySQL。
- `backend/src/middlewares`: JWT 校验、后续权限校验入口。
- `backend/scripts`: 部署、冒烟、端到端联调等开发脚本。
- `backend/hardhat-local`: 本地开发链和 Solidity 合约子项目，用于隔离 Hardhat 3 和后端 CommonJS。
- `backend/sql/init_auth.sql`: 认证与 RBAC 基础表初始化脚本。
- `backend/sql/init_maintenance_v2.sql`: maintenance V2 业务表、签名表、附件表初始化脚本。
- `backend/sql/review_user_permissions.sql`: 审阅用户最终权限效果的视图脚本。
- `backend/storage`: 本地附件、临时文件、隔离区目录。
